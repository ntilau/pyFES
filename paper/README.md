# paper/

IEEE-conference-format paper: *Deep Learning Kernel Based Model Order Reduction of Millimeter-Wave Passive Devices*

## Files

| File | Description |
|---|---|
| `paper.tex` | LaTeX source (IEEEtran, two-column) |
| `paper.pdf` | Compiled PDF (generated by `make` or `pdflatex`) |
| `fig1_results.png` | 6-panel S11/S21 magnitude, phase, error plot (from `plot_surrogate()`) |
| `fig2_uncertainty.png` | S21 predictive ±2σ uncertainty plot |

## Compiling

Requires a standard LaTeX distribution (TeX Live 2020+ / MiKTeX).
Run from this directory:

```bash
# IEEEtran class is included in most TeX distributions.
# If missing: tlmgr install IEEEtran

pdflatex paper.tex
bibtex paper
pdflatex paper.tex
pdflatex paper.tex
```
